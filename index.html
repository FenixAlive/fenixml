Fenixml

Fenixml es un programa que permite ver los archivos cfdi 3.3 en su representación impresa al dar doble click sobre el archivo xml

Fenixml ademas realiza una solicitud de validación al SAT al momento de abrir el xml y muestra su resultado:

Fenixml solo permite hacer una visualización de la información por lo cual no es posible guardar ni imprimir este archivo

Fenixml permite visualizar todos los elementos importantes del xml, ademas de realizar la representación impresa del Qr y la cadena original

Fenixml solo funciona con documentos xml con la versión 3.3 que es la que actualmente se utiliza

Fenixml permite ver y ocultar elementos para una mejor visualización de los datos que se buscan

Fenixml funciona para cualquier tipo de documento: Ingreso, Egreso, Traslado, Pago y Nominas, ademas de poder visualizar cualquier tipo de complemento que se tenga en el xml
